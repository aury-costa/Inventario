# Inventário de Estoque (HTML)
## Como abrir (offline)
1) Extraia o ZIP
2) Abra o arquivo `index.html` no navegador

## Carregar CSV
- Se o arquivo `estoque.csv` estiver na mesma pasta, ele carrega automaticamente.
- Ou use o botão **Carregar CSV**.

## Dicas
- As contagens ficam salvas neste dispositivo (localStorage).
- Use **Exportar** para baixar um CSV com contagens/diferenças/impacto.
